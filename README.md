# LightFields.jl
This julia package provides a novel technique for [Sparse Light Field Reconstruction](https://www.researchgate.net/publication/282403335_Light_Field_Reconstruction_Using_Shearlet_Transform) proposed by S. Vagharshakyan, R. Bregovic and A. Gotchev, using the Shearlet Transform provided by the julia package [Shearlab.jl](https://github.com/arsenal9971/Shearlab.jl).
